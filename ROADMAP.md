# CHTL 项目路线图

本文档旨在跟踪 CHTL 项目的开发进度，并根据 `CHTL.md` 的规范明确各项功能的完成度。

| --- | --- | --- |
| **基础** | | |

| 字面量 (无引号, `""`, `''`) | ✅ 已完成 | |

| 属性 | ✅ 已完成 | |

| 内联样式 | ✅ 已完成 | |

| 上下文推导 (`&`) | ✅ 已完成 | |
