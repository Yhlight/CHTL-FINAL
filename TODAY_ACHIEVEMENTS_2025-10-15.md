## 🎨 功能演示

### 自动类名 + 上下文推导 + 自动 head

**输入 (仅 8 行)**:
```chtl
html {
    body {
        div {
            style {
                .box { width: 100px; }
                &:hover { width: 120px; }
            }
        }
    }
}
```

**输出 (美化的 HTML)**:
```html
<html>
  <head>
    <style>
      .box {
        width: 100px;
      }

      .box:hover {
        width: 120px;
      }
    </style>
  </head>
  <body>
    <div class="box" />
  </body>
</html>
```

**魔法** ✨:
- ✅ 没写 `<head>` - 自动创建
- ✅ 没写 `class="box"` - 自动添加
- ✅ 用 `&:hover` - 简洁优雅
- ✅ 样式自动注入到 `<head>`
