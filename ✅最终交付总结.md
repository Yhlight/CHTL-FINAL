## 🎨 功能演示

### 1. 事件绑定 (&->)

```chtl
{{button}} &-> {
    click: handleClick,
    mouseenter: handleEnter
};
```

### 2. 路由系统 (Router)

```chtl
Router {
    url: "/home",
    page: {{#home-page}},
}
Router { mode: "history" }
```

### 3. 代码格式化 (Formatter)

```bash
# 格式化前
div{class="test"}{text{Hello}}

# 格式化后
div {
    class = test;
}
text { Hello }
```

### 4. 响应式值 ($...$)

```chtl
div {
    class = $boxClass$;
    style { width: $width$ + 20px; }
}
```
