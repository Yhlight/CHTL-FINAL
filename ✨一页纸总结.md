## ✅ 实现的功能

### 1. 事件绑定操作符 (`&->`)
```chtl
{{box}} &-> click: () => { };
{{box}} &-> click, hover: handler;
{{box}} &-> { click: h1, hover: h2 };
```

### 2. 路由系统 (`Router`)
```chtl
Router { url: "/home", page: {{#home}} }
Router { mode: "history" }
```

### 3. 增强Lexer
- 识别 `&->`, `->`, `$...$`
- 支持字符串和数字

### 4. 响应式值 (`$变量$`)
```chtl
div {
    class = $boxClass$;
    style { width: $width$ + 20px; }
}
```
